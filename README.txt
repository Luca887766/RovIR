This is a simple project in with we have to build a site to control a raspberry pi 5 mounted on a 3d printed rover, the rover is equipped with a double camera with Normal vision and Night vision.

To autostart the server on the raspberry we used this tutorial: https://linuxconfig.org/how-to-autostart-bash-script-on-startup-on-raspberry-pi

To eneable the camera we used this tutorial: https://thepihut.com/blogs/raspberry-pi-tutorials/how-to-use-two-camera-modules-with-raspberry-pi-5
